# Texas Communications Unit GitHub Profile

This repository contains shared GitHub configuration and the public organization profile for the **Texas Communications Unit (TX-COMU)**.

The organization profile content is maintained in [`profile/README.md`](profile/README.md) and displayed on the [Texas Communications Unit GitHub organization page](https://github.com/Texas-Communications-Unit).

Learn more about TX-COMU at [tx-comu.org](https://tx-comu.org).
