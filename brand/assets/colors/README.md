# TX-COMU Digital Colors

The files in this directory provide the current centralized digital color tokens used by the TX-COMU website.

## Color Palette

Each approved digital color is shown below with its HEX code.

![TX-COMU digital color palette showing Navy, Blue, Slate, Light, White, and Red](tx-comu-color-palette.svg)

| Color | HEX | RGB |
|---|---|---|
| Navy | `#10233F` | `16, 35, 63` |
| Blue | `#1F5F99` | `31, 95, 153` |
| Slate | `#465466` | `70, 84, 102` |
| Light | `#F4F7FB` | `244, 247, 251` |
| White | `#FFFFFF` | `255, 255, 255` |
| Red | `#D72638` | `215, 38, 56` |

## Available Files

- [tx-comu-colors.css](tx-comu-colors.css) — CSS custom properties
- [tx-comu-colors.json](tx-comu-colors.json) — platform-neutral token data
- [tx-comu-color-palette.svg](tx-comu-color-palette.svg) — downloadable visual reference

These are digital values. Printed output should be proofed for the intended process and material.
