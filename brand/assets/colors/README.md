# TX-COMU Digital Colors

The files in this directory provide the current centralized digital color tokens used by the TX-COMU website.

## Color Palette

Each approved color is shown below with its HEX, RGB, HSL, HSV/HSB, and approximate CMYK values.

![TX-COMU digital color palette showing Navy, Blue, Slate, Light, White, and Red with HEX, RGB, HSL, HSV, and CMYK values](tx-comu-color-palette.svg)

## Approximate CMYK Reference

These CMYK values are mathematical conversions from sRGB and are suitable as general starting points. They are not press-profiled color specifications. Commercial printing should use the printer's ICC profile and an approved physical proof.

## Available Files

- [tx-comu-colors.css](tx-comu-colors.css) — CSS custom properties
- [tx-comu-colors.json](tx-comu-colors.json) — platform-neutral token data
- [tx-comu-color-palette.svg](tx-comu-color-palette.svg) — downloadable visual reference

Pantone, spot-color, and printer-specific matches are intentionally not assigned from screen values alone. Those require a physical color standard and proofing for the intended process and material.
