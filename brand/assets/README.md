# TX-COMU Brand Assets

This directory contains reusable, version-controlled TX-COMU brand assets.

| Directory | Contents |
| --- | --- |
| [logos](logos/) | Approved logo files and usage notes |
| [colors](colors/) | Digital color tokens and visual palette |

Use the assets directly from this repository. Do not recreate the logo from screenshots, trace raster artwork, or substitute sampled colors.
