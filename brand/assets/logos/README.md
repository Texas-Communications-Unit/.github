# Approved TX-COMU Logos

This directory contains the working web assets and the complete set of current source logos maintained in the Google Drive **Texas Communications Unit TX-COMU / Logo** folder.

## Working assets

| File | Dimensions | Intended use |
| --- | ---: | --- |
| [tx-comu-logo.png](tx-comu-logo.png) | 1563 × 1563 | General web, document, and presentation use |
| [tx-comu-github-avatar.png](tx-comu-github-avatar.png) | 460 × 460 | GitHub and other square-profile applications |

Both working PNG files use transparent backgrounds. The full logo is the preferred general-use version. The avatar file reflects the composition currently used for the TX-COMU GitHub organization profile.

When the source-of-truth Drive folder changes, update this directory and its inventory together.
